# Background
- Define um fundo para o elemento
- Sua área de atuação é a caixa toda
- Transparente por padrão

## Exemplos
- Usar cores sólidas
- Usar imagens
- Cotrolar
    - a posição das imagens
    - se elas se repetem ou não
    - o tamanho delas na caixa
- Usar cor e imagem juntas
- Usar cor gradiente
